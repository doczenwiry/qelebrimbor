# qelebrimbor 

---

qelebrimbor started as a toolbox for studying the relationships between ZX-graphs and Surface Code/Lattice Surgery.
It is now a rapidly evolving prototype to solve the problem of transforming a ZX-graph obtained from a quantum circuit into a Blockgraph that can be used by TQEC (Topological Quantum Error Correction).

> **_NOTE:_** This project is under active development and provides no backwards compatibility.
